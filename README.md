# PlatformGame
Jogo de plataforma feito em WindowsForms e que utiliza o padrão de programação estruturada.

O jogo basicamente consiste em fazer Wuo, o nosso personagem principal, pegar 3 chaves
e passar por 3 minigames, ele pega a chave e abre a porta do minigame.
Quanto menor for o tempo que Wuo passar os 3 minigames e quanto mais pontos ele fizer
neles, maior a pontuação no ranking.

Cada minigame tem uma tela de tutorial única, em que se explica o comando e uma contextualização
desse minigame no mundo do jogo. Para o nosso ranking, guardamos os dados de pontuação do usuário,
e salvamos em um arquivo texto, depois, pegamos os dados dessa lista e ordamos pela própria
pontuação.
